
## AutoReport

AutoReport is a Python-based Command Line Interface (CLI) application for generating automated reports from structured data.

## Features

- CLI using Typer
- Project setup
- Organized folder structure
- Easy to extend

## Project Structure

```text
AutoReport/
├── data/
├── charts/
├── templates/
├── reports/
├── tests/
├── main.py
├── pyproject.toml
└── README.md
```

## Technologies Used

- Python
- Typer
